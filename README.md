# sora_design_patterns设计模式

## 项目简介📚
本项目是个人对设计模式的理解，并对每种设计模式做出对应的示例代码，以Java语言作为编程代码，Java版本为17
目的是帮助更多人了解和学习设计模式

## 如何使用
你可以直接在下方的索引中找到想要查看的设计模式，通过后面的链接直接跳转到对应模块内。
每个模块内的客户端代码均是`Main`类，可以通过该类运行获得当前设计模式的结果
